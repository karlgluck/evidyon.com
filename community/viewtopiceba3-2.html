I'm baaack :D

I've been typing up some loose ends the last few days--you know, applying to grad school, making sure I'm registered for classes, trying to figure out how to pay for my education :o)

I am devoting the next two weeks to nothing but Evidyon. As much time as I can, every day, is being spent working on this game (with intermittent exercise so that I stay energized!) My plan is still to get the open version out when I get back to school.

Things are coming along very smoothly--I found more awesome monster meshes to put into the game, have tested many different spell types, and have planned a lot of the mapping utilities. Everything seems to be coming together, but I need to get a feel for what you guys want to see.

As it stands right now, this version of the game just got monsters spawning in it yesterday. They don't do anything yet--that's what I'm working on today. This was a bigger challenge than I had initially anticipated, since I wanted to do it right and make sure it was suitable for planning large maps (which takes a ridiculously long time in the old version) and for handling many different types of monsters (there is only 1 type currently)

If there is enough demand, I hope to release the next version with a single adept-like class as soon as I get the melee monsters spawning correctly and attacking characters. After that, I intend to continue work fixing the bazaar, adding the geosids (since these are absolutely essential to the beta) and fleshing out the classes with more spells.

So--let me know what you want to see :D
