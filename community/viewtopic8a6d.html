Untold wrote:EVERYTHING!

This is what is suppose to be oldschool xen?

It's nothing like xen!

No, this is not supposed to be old-school Xenimus. This is not a Xenimus clone, nor was it ever intended to be a Xenimus clone. Despite this unfortunately common misconception, Evidyon is only intended to capture what we believe old-school Xenimus got right--what made it exciting. We believe that this lies primarily in the fast-paced, immediate-response, player-controlled action of play, and are doing our best to achieve that.

You might fairly respond, "but you haven't managed it." To which the obvious answer is, "we're still in Beta-minus-2: we're working on it."

If you have any thoughts as to how we might improve Evidyon, feel free to post them on the forums. Please do keep in mind, though, that Evidyon is still very much a work in progress, and we can only address one issue at a time--so sometimes things that we want to have have to wait (e.g., a system of item props similar to Xenimus's: we designed one back in May, but won't be able to implement it for a good while yet). In other words, don't be surprised if we say "great idea, but not yet."

--I hope this clears up any misunderstandings you may have had about the project, and I look forward to seeing you around in-game and on the forums.

Best,
Joe M.
