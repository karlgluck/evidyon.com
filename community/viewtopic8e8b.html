Re: What's going on with this update

by Joe M. » Wed Aug 26, 2009 8:46 pm

The server has been taken down until the next update.

I'm not certain how long it will be until it's released (I haven't asked Karl), but here are the changes that you can expect to see:
  • New Server - We finally upgraded to a better server!
  • New Map - The current map was put together to test certain functions of the game. The new map will be built with an eye toward a more playable area.
    • Safe Square
    • Arena Area - for pvp without dropping items or losing alignment
    • NPC guard(s) (maybe)
  • New Spells - and fewer of them, 'cause this is just ridiculous
  • Lower Spawnrate - so that we all have a chance to breathe
  • New and Simpler Geonite System - the current one is overly complicated and doesn't feel right in-game, so we're simplifying things.
  • Humans
