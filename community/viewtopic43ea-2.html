Since you've been waiting a while for an update...

by Karl G. » Tue Oct 14, 2008 12:15 pm

I've decided to release the next update today, even though it is not as complete as I had hoped. I'll be posting the download once I redo the drops again--for some reason, when I import the game project file, it always decides to erase the loot and treasure.

Here are some things it will feature:
  • Fixed that stupid text bug where text would go all wacky sometimes
  • New chat menu and chat log
  • New controls for moving/attacking/blocking (you don't actually block attacks yet--sorry )
  • You can "stop action" to get out of gas
  • Bows work! No arrows though--they just shoot by the way, I need an arrow model!
  • Animation speed is now a factor of your actual ability speed: if you attack slowly, the animation is slower. It looks better and was something that had been bothering me
  • Your player motion is MUCH more fluid--(almost) no more jerking or popping while walking around
  • Screen tearing in fullscreen mode has been fixed
  • Repeatedly moving forward then snapping back when bumping into a monster/tree/mountain has been reduced
  • You shouldn't be able to get stuck in monsters now--I found the bug causing this
  • New character login screen and ability to delete characters
  • Monsters with low HP will flee
  • I adjusted the speed of movement and attacking for monsters and players.
  • Day/Night Lighting

Stuff that I definitely am going to include but haven't had the time to set up yet:
  • Configurable keys for bringing up menus
  • Trading
  • Bazaar
  • Geosids
  • Portals
  • Special item abilities
