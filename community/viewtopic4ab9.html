I'm going to make some "Resource Pack" methods for the editor (Create and Import) that will let you pack up images, textures, models and what-not into a single flat file that can then be used by the editor. This is actually quite easy to do because of the DirectCraft library I wrote--it shouldn't take more than an afternoon. This way, anyone creating resources can just generate one of these Resource Packs, and people can import the pack and already have the models, textures, images, scenery, terrain, etc...set up and ready to go without any more hassle.
