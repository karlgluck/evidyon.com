Upcoming New Feature!

Posted: Sun Mar 11, 2007 3:40 pm
by Karl G.
Okay everyone, Joe and I had a long talk today about how best to get the community more involved in this project. For the reasons stated before, we can't make this game open source--but we can come pretty darn close by making it essentially a "skeleton" game in which all the content is created by data files. The best part is, I already have a working language that can do this! Unfortunately, I have to do some serious clean-up on the code for the client and map editor to include this new feature--but once it's in, you guys won't have to wait for me to add your models to the game or load up new maps in the client.
