Yeah, I talked to him. It might be Vista-related, but I'm running Vista and have sound.

Assuming that your speakers are on and your volume's not muted , check these things:
(a) in the Evidyon folder open the config.ini file and set the sound flag to 1.
(b) if your computer has a sound mixer make sure that the Evidyon volume is not muted. I had mine muted accidentally for a couple weeks before I figured it out.

Also, what I said above: if you're running the game fullscreen and alt-tab, you lose sound.

Hope one of these helps. If not, we'll add it to the bug list.

Actually, the sound thing is my bad. The computers where I did the big "Evidyon shindig" don't have sound cards, so I disabled sound in Evidyon by default and forgot to re-enable it.

Go into Start > Program Files > Evidyon > Edit Configuration and change the 0 below to a 1:

[Audio]
Enable=0


(this file is C:\Program Files\Evidyon\config.ini)
