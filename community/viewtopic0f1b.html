New Editor - Almost here!

Posted: Thu Jan 10, 2008 5:22 pm
by Karl G.
The next version of the world editor is almost ready to be released! This will be one of the last major feature updates I do before getting back to cracking at the client. The "resource pack" functionality is almost done--adding new content to your map will now be a one-click process. These packs will add images, textures, meshes, skins, scenery and terrain to your world automatically.

All that's stopping me from releasing this thing today is that I want to get some content packs together for you to use. I'm going to organize, re-scale and set up scenery for all of the free content I've gathered from across the web, along with the content I've bought specifically for PV. Finally, I'm going to update the quick-start guide for the new editor and change some things in it based on feedback.

The new editor will not have rules implemented. However, it will have default texture ranges.
