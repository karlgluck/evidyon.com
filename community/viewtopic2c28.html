I'm using this post as a place to publicize features, notes, a to-do list and ideas as I get closer to releasing the alpha.

Features to Implement

  • Packet encryption

  • Build spellcasting/attacking animations
  • More monster types
  • Fill out the list of player races/classes
    • Make new skins for different races
  • Party system
  • Trade window

  • Storage
  • Ability to delete characters
  • Improved log-in screen
  • Configurable inventory/stats/spells screen keys (right now they're permanently bound to i/s/k)

  • Monster spawn points and spawn rates

  • Experience points based on levels

  • Level-up
  • Key binding window
  • Make monsters increase in difficulty as level is adjusted
  • Monster behavior based on range to target (ex. ranged attacks)
