If you're new, check out the basic instructions.

Triggering Quests
Right-click an NPC. Easy as that.

Your Quest Log
Right-click yourself to bring up the quest log. It will show you the amount of time remaining if the time is less than 24 hours. Clicking a quest brings up the description and allows you to give up.

Current Quests
There are 4 quests in the game right now. They are just for testing out the system; Beta will feature many more.

by Karl G. » Fri Jun 05, 2009 8:19 am

Huh, it seems like the game isn't copying over the file it downloaded. Go into your Evidyon directory and look for "evidyon.gme" and "~evidyon.gme". The second one (with the ~) is the new version it downloaded. Delete the old version and remove the ~ from the new version so that it is called the same thing. IF there are other files with a ~ in front (perhaps evidyon.exe?) change them as well.

I will also upload a new version of the installer with the latest game file, so if that doesn't work you can try with an updated install.
