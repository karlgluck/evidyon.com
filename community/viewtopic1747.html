Evidyon Pre-Beta RELAUNCH TOMORROW! (Monday Night ~9:30 PM)

by Karl G. » Sun Aug 30, 2009 3:02 pm

I've been working hard this week to get all the changes in place, and things are shaping up very well. Expect to see:
  • New, redesigned spell set
  • Levels 1-15 available
  • Town guards!
  • Safe zones
  • More simple, more fun Geosid behavior
  • BRAND NEW SERVER MACHINE! Goodbye, lag.
  • Redesigned chaos
  • Balanced PvE and (hopefully) PVP
  • Valtus Mederi Rings (keep you from respawning on death)
  • Consumables
  • Portals
  • Improved server stability

Note: we are still testing equations and balance at this point. We hope to lift the level cap and move to a more finalized version by the end of this week.

Karl G.

Yes, items dropping on death works the same way--I have added the code for something to change that, but it hasn't been tested yet.
