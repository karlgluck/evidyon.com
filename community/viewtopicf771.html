Countdown to Alpha!

by Karl G. » Wed Aug 13, 2008 7:27 am

I'm using this post as a place to publicize features, notes, a to-do list and ideas as I get closer to releasing the alpha.


To-Do List
  • Formal bug-tracking on web
  • Form customer service group
  • New forum skin

Features to Implement
  • Stats window
  • Music
  • Consumable items
  • Packet encryption
  • Resource obfuscation
  • Add more items, spells
    • Implement new spell effect types
  • Equipped items should be attached to avatar
  • Build spellcasting/attacking animations
  • More monster types
  • Fill out the list of player races/classes
    • Make new skins for different races
  • Party system
  • Trade window
  • Item weight
  • Storage
  • Ability to delete characters
  • Improved log-in screen
  • Configurable inventory/stats/spells screen keys (right now they're permanently bound to i/s/k)
  • Block text as a TT font for use in-game

Features to Debug
  • Newly available effects on level-up don't show up in key binding window until you log out and back in
  • Animated meshes won't work using D3D mixed mode on some lower-end hardware

Implemented Features
  • Aggressive monsters (new!)
  • Equipping items works correctly (new!)
  • Monster spawn points and spawn rates
  • Sound effects
  • Monsters drop assignable, random items on death
  • Experience points based on levels
  • Character death/re-spawning
  • Level-up
  • Key binding window
  • Make monsters increase in difficulty as level is adjusted
  • Monster behavior based on range to target (ex. ranged attacks)

    Requested Features & Design Points
    • "safe zone"
    • "Bazaar" system
    • Bounty system

    Miscellaneous
    • (none)
