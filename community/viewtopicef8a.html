steps on nice sketches-

first use pencil to sketch it out (mainly to erase)

after that use a heavy ballpoint pen to outlined said sketch.

after it is outlined use eraser to erase the pencil (makes it look cleaner)

there yah go best results form sketches, and if you really want to make changes trace the original inked version in pencil so if it is critiqued you can make the changes as said. these are all optional.

thank you for participating.
