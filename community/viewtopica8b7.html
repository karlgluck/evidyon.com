Re: Evidyon Pre-Beta Opening This Weekend

Posted: Sun Aug 23, 2009 11:05 am
by Joe M.
As y'all can tell, there are a few small problems with the release. Naturally, we're working on getting these problems fixed first of all. I haven't yet talked to Karl this afternoon, but I'll call him in an hour or so to figure out where we're at and let y'all know.

As for the problems themselves:
* As has been said, try setting the ForceSoftwareRendering flag to 1.
* Make sure that you're viewing hidden files (in the "Folder Options" dialogue, I believe), and run the launcher. It may be having trouble overwriting the current file when it downloads a new one. Make sure that after it downloads an update, that it does *not* leave a "~evidyon.exe" file in your directory. If it does, delete the (hidden) "evidyon.exe", and rename "~evidyon.exe" to "evidyon.exe" ; then run "evidyon.exe" directly, instead of using the launcher.

Like I said--I'll post the minute I have more information.
