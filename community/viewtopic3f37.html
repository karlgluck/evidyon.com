The definitive ultimate texture contribution thread

by thadiusofx3 » Wed Feb 28, 2007 12:04 pm

Listen up guys, we need all of the textures in the texture pack replaced with ones that look similar.

Here are some guidelines for textures:


1. Try to make the new texture the same resolution (size) as the old one. ex. if the old texture is 128x128 make the new one that size too
2. Try to get relatively close to the same design as the old texture with the new one. We want stuff to look similar but no exactly the same.
3. Some textures like bauul.png will need to be replaced later when we make the models for it. So skip those right now.
4. All submitted texture need to atleast be in PNG format. If you use photoshop or something similiar, also upload the RAW format image to make it easier on Karl to edit.
5. NO COPYRIGHTED OR LICENSED MATERIAL! Either find completely royalty/license free textures or make them yourself. We don't want any legal troubles.

Once you complete a texture, upload it to the ftp server in /3d modeling/textures/replacements/ folder and make a post with the new texture image in it, and also the name so I can change the list to reflect progress.


Red = Incomplete
Blue = Complete

by Karl G. » Wed Feb 28, 2007 3:07 pm

I'd just like to add that if you use Photoshop or other image editing software to make the images before exporting to PNG, please upload the raw source images, too. For example, the grass textures are all linked together (grass alone, over dirt, etc.) and it would be much easier to edit if they're available in a layer-based format.

Also: please note that many of the images are actually four different images composed into a single texture. When replacing them, please consider each quadrant to be a separate image and create the replacement accordingly (no overlaps between quadrants)
