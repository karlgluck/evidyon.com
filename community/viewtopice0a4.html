OK Folks I'm back!

by Karl G. » Fri Mar 13, 2009 2:00 pm

I'm going to be attacking some bugs this afternoon, and addressing some of the littler things that have piled up. So far I've finished the following:
  • I think I fixed the BSOD bug!
  • Some sneaky memory leaks have been patched thanks to a kickass tool I found
  • There are now icons for rings, gloves and potions
  • Inventory order is now saved on log in/out. It also works like you would expect, with most recently picked up items at the end. If you click an item and it can't be equipped, it will go to the end of the list.
  • Green potions now work without having to select a target for the spell
  • I added a 'red potion'
  • All items should now stack correctly
  • -respawn underground will put you back on the main map
  • Merchant range has been increased
  • Items not showing up in the inventory should be fixed
  • "Freeze zone" should be gone
  • Collisions with objects should be more intuitive now
