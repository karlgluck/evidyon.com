After wrestling with the server, I finally got it configured correctly so that the game is playable. That took all my time this weekend (that I wasn't doing valentine's day stuff ) so this update only has a couple of new things. One of them is HUGELY important, though: the auto-updater has been enabled. I can push quick updates out to you as soon as I make them, so expect incremental updates this week enabling some of the features I was working toward today.

A side note: the server does NOT have your characters on it and is basically the same as last version. Once poorserver (the old one) comes back up, I *will* be restoring your characters, so don't bother leveling right now. If you want to log in and check out the lag time/responsiveness, that's pretty much what this update is for.
