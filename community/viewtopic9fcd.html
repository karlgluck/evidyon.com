How to Install
  1. Download the map tools ZIP file (pvmaptools_alpha0.2.zip) and the content ZIP file (xenmedia.zip)
  2. Unzip the pvmaptools_alpha0.1.zip file to a directory called "map tools"
  3. Unzip xenmedia.zip to the "map tools/Media" directory (it will be created when you unzip the first file)
  4. Say "yes" to overwrite the media.dat and maped.dat files
  5. Read the readme file for whichever program you want
  6. Run the program

Resources
  • download - The latest version of the tools (6/27)
  • Some content:
    • Copy the contents of the Xenimus folder into the /Media/ directory
    • download - Abridged version of the Xenimus media
  • download - The Microsoft SxS libraries (you only need to download this once; its resolves the "application configuration incorrect" error)
  • download - The latest DirectX


Bugs and Suggestions
