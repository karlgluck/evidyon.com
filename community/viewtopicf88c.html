[OLD] How to fix "d3dx9_38.dll not found" error

by Karl G. » Tue Aug 26, 2008 7:18 pm

Okay so the problem here is with the new installer; it doesn't include the DirectX configuration program. To fix this, download this package:

http://www.unseenstudios.com/evidyon/do ... vidyon.zip

Extract and run "DXSETUP.EXE". You can delete the package when you're done.
