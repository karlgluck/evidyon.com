What's going on with this update

Posted: Sun Aug 23, 2009 5:47 pm
by Karl G.
First, I want to thank everyone for sticking with me and showing up last night for the first test of the new version. There were obviously some issues that need to be resolved, and I will be coding all day tomorrow (I spent today recovering from lack of sleep over the last week and doing minor tweaks).

Tonight, I am playing the game with a few friends here at my place to get everything in place for a re-launch of this version. So far I have fixed:
- several crash bugs (I've played for an hour or so without problems)
- fixed geosids (their bonuses will now show up!)
- slowed monsters down, sped players up so that the game feels much more responsive
- repeatedly clicking on monsters to attack them doesn't cause your attacks to keep resetting
- AOE spells are fixed
- the launcher can now update your files
- DOT spells will now work correctly on any target

So here's my plan: I'll release a new version with all the fixes tonight. You'll need to re-download the installer.

Because we are still trying to figure out how to balance everything, make sure the game is stable and make it more fun (!!) we need you guys to help us test.

To this end, we will be doing reset-tests of the game until next week. Each day expect to log in to a completely wiped server with fixes based on your feedback from the previous version. You can get in, play, test things out and let me know what needs to be fixed here on the forum. Once Pre-Beta starts, we will start the "official" version where you get to keep your characters, and we will start adding content/maps/spells/etc and increasing the level cap as fast as we can.

Great to hear that everyone had a good time!
