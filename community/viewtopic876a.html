Thanks for the comments everyone! I've jumped on a computer to respond.

JSunJShineR wrote:Hey Karl, I love it man. Are those the only spells though?

So I only actually SHOWED you about 1/5 of the spells. If you look at that list, those are all of the spells in the game right now. However, I'd like to make clear that content for this game is INCREDIBLY easy to add. Once you know what kind of spell you want to make, it literally takes less than 5 minutes to add it to the game (including logging into the server and testing it out, then going back and tinkering a few times). The same goes for new items, changing character sizes, adding new kinds of monsters, changing which items drop from various monsters, how those monsters attack and how they behave--everything is completely configurable. So the hard part is the design; which is exactly how it should be. Right now I haven't done thorough design, however, because I'm still having issues getting it to run properly on different hardware. For example, it BSoD's my girlfriend's laptop when she logs into the world...

JSunJShineR wrote:Only thing I can point out that I don't like to much is the character size, it seems a little big for the world

Yep. The character I'm playing with is also a "Giant", but since there are no other players it's hard to tell =)

Rock solid 60 FPS. However, my laptop REALLY doesn't like running multiple monitors while recording at 30 FPS (which drops the frame rendering rate, but not the game speed, making it look choppier) and running the client and server simultaneously. When not recording, it runs great though!
