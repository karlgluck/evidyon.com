Huh, it seems like the game isn't copying over the file it downloaded. Go into your Evidyon directory and look for "evidyon.gme" and "~evidyon.gme". The second one (with the ~) is the new version it downloaded. Delete the old version and remove the ~ from the new version so that it is called the same thing. IF there are other files with a ~ in front (perhaps evidyon.exe?) change them as well.
