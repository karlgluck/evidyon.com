Just uploaded a rough recreation of "relax."

link: http://www.mydatabus.com/public/Test2/z/relax_rough.mp3

Notes:

1) This is a recreation, the melody and instrumentation aren't exactly the same. (and if you listen close the guitar part isn't either)

2) This is "rough" because I left my main hardware up at college over break. The family comp I'm working on now can barely handle 3 instruments sampled at 44k.

When I get back from break I'll release a final version of this that will be a lot better. (Better orchestration, higher quality sounds, some dynamic variation, and a little longer). I'd also like anyone's opinion on the fret/string noise. I like it because it's a more authentic* sound, but it might be weird to have that in a game.

Since this isn't the final version I just left it in .mp3. If anyone would like the midi or FL file for this to tweak, let me know and I'll upload them.

*all of relax is in a-minor and can technically be played on guitar without moving up the strings (so no fret noise), but the VST plugin I'm using isn't advanced enough to know that (and I doubt any program is).
