by Karl G. » Sun Jan 06, 2008 9:08 am

As style goes, have a look at these:

http://www.basilstudio.com/Props.html

I bought these medieval models a while back and I'm going to be distributing it as soon as I find a way to pack up the resources into a format that fits the EULA--it says I can't distribute the models in their original format. But actually, this gave me a GREAT idea!

I'm going to make some "Resource Pack" methods for the editor (Create and Import) that will let you pack up images, textures, models and what-not into a single flat file that can then be used by the editor. This is actually quite easy to do because of the DirectCraft library I wrote--it shouldn't take more than an afternoon. This way, anyone creating resources can just generate one of these Resource Packs, and people can import the pack and already have the models, textures, images, scenery, terrain, etc...set up and ready to go without any more hassle.

That's on the to-do list now




Aaaaanyway, we really need more outdoor scenery models and textures. Trees, plants, blades of grass, funny swamp rushes, vines on sticks, crops, cacti, etc...the model pack had none and none of the ones I've been able to find online are any good at ALL so I've been using Xenimus's models....which are not wonderful either.
